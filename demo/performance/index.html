Interpret & Initialize
This calls render(container, html`…`), where the expression html`…` has never been executed before and the container has never been initialized.
Initialize Only
This calls render(container, html`…`), where the expression html`…` has been executed before, but the container has never been initialized.
Update Only
This calls render(container, html`…`), where the expression html`…` has been executed before and the container has already been initialized.
Explainer

Each row in each group is run in an iframe at a unique url — one-by-one. Dots represent percentiles (20th-80th). The term interpretation is the parsing of markup as instructions to create dom. The term initialize refers to populating a given container with dom nodes which have properties bound. The term update is just the act of binding new values to existing dom in an already-initialized container. See PERFORMANCE.md for additional details.
